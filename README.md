# Nothing(PC) <sup>unofficial</sup>
Nothing (PC) Unofficial is a lightweight, native Windows utility (C++) that integrates your Nothing and CMF ecosystem earbuds directly into your system.

## Supported devices
* CMF Buds / CMF Buds Pro / CMF Buds Pro 2 _(donphan)_

* Nothing Ear (a) _(cleffa)_

* Nothing Ear (open) _(flaaffy)_

* Nothing Ear (1) / Ear (2) / Ear (3) _(crobat / corsola)_

## Features

* Battery percentage (Case/Buds)
* Bass Enhance and ANC settings
* Find my Earbuds
* Low Latency Mode

## Requirements

* Built-in or external USB Bluetooth adapter supporting BLE
* Windows 10 (Build 19041 and above) or Windows 11
